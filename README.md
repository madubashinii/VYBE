A **full-stack workout tracking app** built with **Next.js** (frontend) and **Node.js/Express** (backend). Effortlessly log workouts, track progress, create custom plans, and stay motivated with reminders and streaks.  

---

## Key Features

- **User Authentication**: Secure login using **JWT** or **OAuth**  
- **Workout Logging**: Track sets, reps, duration, calories burned, and more  
- **Progress Dashboard**: Visualize your journey with charts, streaks, and statistics  
- **Workout Plans**: Browse or create **HIIT**, **strength**, **yoga**, and custom plans  
- **Reminders & Motivation**: Daily alerts to keep you on track  
- **Social Features (Optional)**: Add friends, share achievements, and compete on leaderboards  
- **Admin Panel**: Manage users, plans, and content efficiently
